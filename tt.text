  1.
  直接線上匯出（不需安裝，最快）

  把 classDiagram ... 的內容（不
  含外層的三個反引號）貼到
  mermaid.live，右側直接有 PNG /
  SVG 下載按鈕。

  ---
  2. VS Code 外掛

  安裝 Markdown Preview Mermaid
  Support，開啟 ClassDiagram.md
  預覽後，右鍵圖片可另存 PNG。

  ---
  3. mmdc
  CLI（本地指令，適合批次）

  需要 Node.js：

  npm install -g
  @mermaid-js/mermaid-cli
  mmdc -i .res/ClassDiagram.md
  -o .res/ClassDiagram.png -w
  2400

  會直接在 .res/ 產出 PNG。-w
  控制寬度，Class Diagram
  類別多建議給大一點（2000+）。
